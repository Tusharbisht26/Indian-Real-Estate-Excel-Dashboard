# Indian Real Estate Analytics Dashboard Using Excel

## Project Overview
This project is an interactive Excel dashboard created to analyze Indian real estate market data. The dashboard provides insights into property pricing, rental yield, ROI, market trends, and property distribution across different cities and property types.

The project demonstrates data cleaning, exploratory data analysis (EDA), KPI reporting, dashboard development, and business insight generation using Microsoft Excel.

---

## Business Problem
Real estate investors and agencies require data-driven insights to understand:
- Property pricing trends
- ROI performance
- Rental yield opportunities
- Market demand
- Property status distribution
- City-wise investment potential

This dashboard helps stakeholders make better investment and business decisions.

## Data Cleaning Process
The following cleaning steps were performed before analysis:

- Removed duplicate records
- Standardized inconsistent date formats
- Checked and validated missing values
- Removed unnecessary columns
- Verified data consistency
- Performed outlier analysis using the IQR method

## Outlier Analysis
Outliers were identified using the Interquartile Range (IQR) method.

Formula used:

IQR = Q3 - Q1

Outlier Condition:
- Below: Q1 - 1.5 × IQR
- Above: Q3 + 1.5 × IQR

Outlier analysis was performed on:
- Property Price
- Rent
- Area
- ROI
